## AI Image Generator

This is an AI-based image generator project built using the **Hugging Face API**.

⚠️ **Note:** Hugging Face provides only a limited number of models for free use.

### Requirements
To run this project locally on your machine, you must:
- Create an account on [https://huggingface.co](https://huggingface.co)
- Generate a Hugging Face access token

### Important
Without a valid Hugging Face account and API token, the project will not run locally.
